# Recruitment Plan — Coquitlam / Lower Mainland BC

Goal: **5–10 participants** with arthritis, tremor, or age-related fine motor loss, plus **2–3 caregivers** for observation interviews.

## Inclusion (broad, non-clinical)

- Age 55+ **or** self-reported arthritis / hand tremor affecting daily tasks
- Able to give informed consent (or caregiver-assisted consent)
- Comfortable sitting at a table for 30 minutes

## Exclusion

- Acute hand injury or physician-ordered immobilization
- Photosensitive epilepsy (flashing UI — rare)
- Unable to understand stop/confirm commands even with caregiver help

## Recruitment channels

| Channel | Action |
|---------|--------|
| Local seniors’ centre (Coquitlam) | Offer 45-min demo slot; bring printed one-pager |
| Arthritis Society BC events | Booth demo + sign-up sheet |
| Family / neighbours | Grandparent cohort (your origin story — authentic outreach) |
| School science fair judges | Invite as observers; separate from clinical cohort |
| Physiotherapy student clubs (UBC/SFU) | Observers + referral only — not clinical endorsement |

## One-pager bullet points

- Free research demo — **not** a sold medical device  
- Helps reach bottle, remote, pills on a table  
- Voice or big buttons — no joystick skill needed  
- You control when the arm moves (yes/no)  
- ~30 minutes, snacks provided  

## Session schedule template

| Week | Sessions | Target |
|------|----------|--------|
| 1 | 2 | Pilot facilitator only (debug protocol) |
| 2 | 3 | Early arthritis profile |
| 3 | 3 | Severe profile + panel-only block |
| 4 | 2 | Caregiver + dual participant |

## Ethics note

For school science fair: teacher/supervisor review + parental consent if minors assist as facilitators only. For publication or hospital partnership: formal REB/IRB required — this plan is **not** a substitute.

## Success metrics

- ≥20 trials per condition in `lab_trials.jsonl` for manual vs dual_perception  
- ≥70% task success on ADL button path (moderate profile)  
- Qualitative: majority pick voice or panel as preferred input  

## Materials to bring

- Printed `docs/USER_TESTING_KIT.md` consent section  
- Tape measure for final-distance cm  
- Backup: `python main.py --simulation` if USB fails
